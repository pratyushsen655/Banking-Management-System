# Legacy Banking Files

This folder contains the original console-based banking system files that were in the repository before the comprehensive Banking Management System was implemented.

## Files

- **BankingSystem.java** - Original console-based banking application with basic account management
- **Login.java** - Original Swing-based login interface (incomplete implementation)
- **Signup.java** - Original signup form interface

## Note

These files are kept for reference purposes. For the full-featured Banking Management System with both desktop and web interfaces, please use:
- **Desktop Application:** `../desktop/`
- **Web Application:** `../web/`

See the main [README.md](../README.md) for complete documentation and getting started instructions.
